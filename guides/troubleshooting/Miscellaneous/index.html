Common Bugs and Their Fix

A guide that covers how to fix the most common bug you may encounter while using the Software.

After pressing the Insert key, my PC restarts automatically. What should I do?

Ensure your game is set to windowed or borderless mode.

I'm unable to click anything in the menu. What should I do?

Go to Windows display settings and set the scaling to 100%.

After pressing Insert, I hear a beep but the menu doesn't appear. What should I do?

Press F1 to open the menu. If nothing appears you will hear a sound after pressing Insert

The software disappears during gameplay, and my PC restarts. What should I do?

Update your GPU drivers and increase your virtual memory .

the ESP is lagging what should i do??

For Windows Defender (Recommended)

  • Download Defender Control.
  • Disable Windows Defender then Launch dControl.
  • Press "Disable Windows Defender".
  • Once done, you are good to go.

Disable Windows Game Mode

  • Open Windows Settings.
  • Navigate to Game -> Game Mode.
  • Toggle the "Game Mode" switch to the Off position.

Disable GPU Hardware acceleration

  • Open Windows Settings.
  • Navigate to System -> Display -> Graphics.
  • Toggle the "Optimization For Windowed Games" switch to the Off position.
