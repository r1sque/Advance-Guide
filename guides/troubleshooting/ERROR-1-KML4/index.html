How To Fix ERROR 1 KML4

A guide that covers how to fix ERROR 1 KML4.

1. Disable core isolation

All in one Disabler (Recommended)

This method will automatically disable Core Isolation and other features.
  • Download and extract the contents of the .rar file.
  • Execute all the files within the extracted folder.
  • Once done, Reboot your PC.

Manual Disabler

  • Open Windows Security settings.
  • Navigate to Device Security -> Core Isolation.
  • Toggle the "Memory Integrity" switch to the Off position.
  • Restart the system for the changes to take effect.

2. Uninstall FACEIT / VANGUARD anticheat

All in one Uninstaller (Recommended)

This method will delete any trace of the software installation from your computer.
  • Download Revo Uninstaller. (Portable Version Recommended)
  • Launch Revo Uninstaller and wait for the software to scan all your Installed Apps.
  • Once done, look for "Faceit Anticheat" / "Riot Vanguard", right-click it, and select "Uninstall."
  • Uncheck the "System Restore" option, then click "Continue" to start the uninstallation process.
  • Afterward, click "Scan" wait for it to finish, then select my computer and click "Delete."

Manual Uninstaller

  • Open Windows Security settings.
  • Navigate to Apps -> Installed Apps.
  • Search for "Faceit Anticheat" or "Riot Vanguard".
  • Uninstall the selected anticheat software.
