Loader:

  • Open the launcher and select the game ID you purchased.
  • Disable all antivirus software, including your firewall, Windows Defender, Core Isolation, and Memory Integration in Windows Defender.
  • Use Defender Control to completely disable Windows Defender. Download Here
  • Close all game launchers (e.g., BSG, STEAM, EA, BATTLENET, GAMEPASS, etc...).
  • Run the loader as Admin, select the game, enter your key, and press Enter.
  • After the loader finishes processing, a notification will appear. Then, run the game, switch to "windowed" mode in the graphics settings, and press Insert.
